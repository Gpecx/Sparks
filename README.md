# SPARK ⚡

O SPARK é uma plataforma de aprendizado gamificada desenvolvida em Flutter. O objetivo principal do projeto é transformar o estudo de normas técnicas de segurança e engenharia (como NR-10 e NR-35) em uma experiência dinâmica, interativa e menos engessada para os profissionais da área.

A aplicação utiliza elementos de jogos e uma interface imersiva para manter o engajamento do usuário durante a jornada de capacitação.

## 🚀 Funcionalidades Principais

* **Trilha de Aprendizado:** O mapa de módulos possui um design inspirado em uma "Placa de Circuito", com animações de fluxo de energia que acompanham o progresso do usuário.
* **Motor de Quizzes (Quiz Engine):** Sistema de desafios com feedback imediato. O usuário gerencia uma barra de "bateria" (energia) e sofre penalidades visuais (como um efeito de *Glitch* na tela) em caso de erros.
* **Sistema de Gamificação:** * Acúmulo de *Spark Points*.
  * Leaderboard (Ranking) para incentivar o engajamento.
  * Cartão de Bolso 3D, funcionando como uma credencial digital do técnico.
* **Identidade Visual:** UI/UX com estética cyberpunk/educativa, utilizando fundos escuros, gradientes radiais e elementos em neon.

## 🛠️ Tecnologias Utilizadas

* **Framework:** [Flutter](https://flutter.dev/)
* **Linguagem:** Dart
